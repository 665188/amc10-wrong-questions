# 2016 AMC 10A — Problem 20

> 正确答案：**B** ⬅ 我做错了

## 题目

**English:**

For some particular value of $N$, when $(a+b+c+d+1)^N$ is expanded and like terms are combined, the resulting expression contains exactly $1001$ terms that include all four variables $a, b, c,$ and $d$, each to some positive power. What is $N$?

**中文翻译：**

对于某个特定的 $N$，展开 $(a+b+c+d+1)^N$ 并合并同类项后，恰好有 $1001$ 个项同时包含所有四个变量 $a, b, c, d$，且每个变量的指数至少为 $1$。求 $N$。

## 选项

$$\textbf{(A) }9 \qquad \textbf{(B) } 14 \qquad \textbf{(C) } 16 \qquad \textbf{(D) } 17 \qquad \textbf{(E) } 19$$

## 解题思路

### 步骤 1：理解「包含所有四个变量」

每个「包含所有四个变量」的项形如：

$$a^i b^j c^k d^l \cdot 1^m$$

其中 $i,j,k,l \geq 1$（至少一次幂），$m \geq 0$，且：

$$i + j + k + l + m = N$$

### 步骤 2：转化为组合计数

令 $i' = i-1$，$j' = j-1$，$k' = k-1$，$l' = l-1$（所有 $\geq 0$），则：

$$i' + j' + k' + l' + m = N - 4$$

这是将 $N-4$ 个无差别对象分配到 $5$ 个类别（$i',j',k',l',m$）中。

**星棒法（Stars and Bars）：**

$$\text{项数} = \binom{(N-4) + 5 - 1}{5 - 1} = \binom{N}{4}$$

### 步骤 3：求解方程

$$\binom{N}{4} = 1001$$

$$\frac{N(N-1)(N-2)(N-3)}{24} = 1001$$

$$N(N-1)(N-2)(N-3) = 24024$$

试算：

| $N$ | $\binom{N}{4}$ |
|-----|----------------|
| 13 | 715 |
| **14** | **1001** ✓ |
| 15 | 1365 |

$$\therefore N = 14$$

### 验证

$$14 \times 13 \times 12 \times 11 = 24024$$

$$24024 \div 24 = 1001 \quad\checkmark$$

## 知识点

- 多项式展开（multinomial expansion）
- 星棒法 / 隔板法（stars and bars）
- 组合恒等式
- 变量代换技巧

## 易错点

- ❌ 忘记每项必须包含**所有四个**变量（每个至少一次）
- ❌ 把公式错记为 $\binom{N+4}{4}$ 或 $\binom{N-1}{4}$
- ❌ 把 1001 因式分解错（$1001 = 7 \times 11 \times 13$）
- ❌ 算到 $N=14$ 但误选其他选项
