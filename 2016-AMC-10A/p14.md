# 2016 AMC 10A — Problem 14

> 正确答案：**C** ⬅ 我做错了

## 题目

**English:**

How many ways are there to write $2016$ as the sum of twos and threes, ignoring order? (For example, $1008\cdot 2 + 0\cdot 3$ and $402\cdot 2 + 404\cdot 3$ are two such ways.)

**中文翻译：**

将 $2016$ 写成若干个 $2$ 和若干个 $3$ 的和，不考虑顺序，有多少种写法？（例如 $1008 \times 2 + 0 \times 3$ 和 $402 \times 2 + 404 \times 3$ 是其中两种。）

## 选项

$$\textbf{(A)}\ 236\qquad\textbf{(B)}\ 336\qquad\textbf{(C)}\ 337\qquad\textbf{(D)}\ 403\qquad\textbf{(E)}\ 672$$

## 解题思路

### 转化为方程

设用 $a$ 个 $2$ 和 $b$ 个 $3$（$a, b$ 为非负整数），则：

$$2a + 3b = 2016$$

### 解方程

$$2a = 2016 - 3b \implies a = 1008 - \frac{3b}{2}$$

$a$ 是整数 ⟹ $b$ 必须是偶数。

令 $b = 2k$（$k \geq 0$），则：

$$a = 1008 - 3k$$

$a \geq 0 \implies k \leq 336$

$$\therefore k = 0, 1, 2, \ldots, 336$$

共有 $336 - 0 + 1 = 337$ 种写法。

### 验证

| $b$ | $a$ | 总和 |
|-----|-----|------|
| $0$ | $1008$ | $2016$ ✓ |
| $6$ | $999$ | $2016$ ✓ |
| ... | ... | ... |
| $672$ | $0$ | $2016$ ✓ |

## 知识点

- 丢番图方程（linear Diophantine equation）
- 数列/计数
- 转换思维（将组合问题转为方程求解）

## 易错点

- ❌ 忘记 $a, b$ 必须是非负整数
- ❌ 没有注意到 $b$ 必须是偶数（因为 $2016$ 是偶数，$3b$ 奇偶性取决于 $b$）
- ❌ 计算 $k$ 的范围时偏移了一个（应该是 $337$ 而不是 $336$）
