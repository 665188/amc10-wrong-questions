# 2016 AMC 10A — Problem 23

> 正确答案：**A** ⬅ 我做错了

## 题目

**English:**

A binary operation $\diamondsuit$ has the properties that $a\,\diamondsuit\, (b\,\diamondsuit \,c) = (a\,\diamondsuit \,b)\cdot c$ and that $a\,\diamondsuit \,a=1$ for all nonzero real numbers $a, b,$ and $c$. (Here $\cdot$ represents multiplication). The solution to the equation $2016 \,\diamondsuit\, (6\,\diamondsuit\, x)=100$ can be written as $\tfrac{p}{q}$, where $p$ and $q$ are relatively prime positive integers. What is $p+q?$

**中文翻译：**

二元运算 $\diamondsuit$ 满足：对任意非零实数 $a,b,c$，有 $a\diamondsuit(b\diamondsuit c) = (a\diamondsuit b) \cdot c$，且 $a\diamondsuit a = 1$。（$\cdot$ 表示普通乘法。）方程 $2016 \diamondsuit (6\diamondsuit x) = 100$ 的解可写为 $\frac{p}{q}$（$p,q$ 互质正整数）。求 $p+q$。

## 选项

$$\textbf{(A) }109\qquad\textbf{(B) }201\qquad\textbf{(C) }301\qquad\textbf{(D) }3049\qquad\textbf{(E) }33,601$$

## 解题思路

### 步骤 1：推导运算形式

从已知条件出发：

**条件 1：** $a \diamondsuit a = 1$

**条件 2：** $a \diamondsuit (b \diamondsuit c) = (a \diamondsuit b) \cdot c$

猜测 $a \diamondsuit b = \frac{a}{b}$。验证：

- $a \diamondsuit a = \frac{a}{a} = 1$ ✓
- $a \diamondsuit (b \diamondsuit c) = a \diamondsuit \frac{b}{c} = \frac{a}{b/c} = \frac{ac}{b}$
- $(a \diamondsuit b) \cdot c = \frac{a}{b} \cdot c = \frac{ac}{b}$ ✓

两条性质均满足！所以 $\diamondsuit$ 就是除法。

> 注：也可推出 $a \diamondsuit b = \frac{a}{b}$ 是唯一解。证明思路：令 $b=1$ 可推 $a\diamondsuit 1 = a$，再推出一般形式。

### 步骤 2：解方程

$$2016 \diamondsuit (6 \diamondsuit x) = 100$$

$$2016 \diamondsuit \frac{6}{x} = 100$$

$$\frac{2016}{6/x} = 100$$

$$\frac{2016x}{6} = 100$$

$$336x = 100$$

$$x = \frac{100}{336} = \frac{25}{84}$$

### 步骤 3：求 $p+q$

$p = 25,\; q = 84$，二者互质（$25=5^2$，$84=2^2 \times 3 \times 7$）。

$$p + q = 25 + 84 = 109$$

## 知识点

- 函数方程（functional equations）
- 二元运算的推导
- 代数化简

## 易错点

- ❌ 以为 $\diamondsuit$ 是乘法（$a\diamondsuit a = a^2 \neq 1$）
- ❌ 推导过程中忘记 $\diamondsuit$ 不满足交换律
- ❌ $2016/(6/x)$ 化简时搞错分数层级
- ❌ 最后一步忘记约分，直接用 $100/336$ 求 $p+q$
